**Prerequisites:** Node.js, npm

1. Clone the repo

   ```bash
   git clone https://github.com/auauron/SE-Testing.git
   cd client
   ```
2. Install dependencies

   ```bash
   npm install
   ```

3. Run Storybook

   ```bash
   npm run storybook
   ```

4. Open http://localhost:6006
